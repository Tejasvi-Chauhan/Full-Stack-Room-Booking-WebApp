# A Full-Stack Room Booking web Application

This project is a full-stack rental booking web application inspired by Airbnb. It allows users to explore listings, view detailed descriptions, and perform CRUD operations on rental properties. It focuses on practical implementation of web development concepts using modern technologies.

##  Features

-  View listings with images, location, description, and price
-  Add new listings using a validated form
-  Edit existing listings with updated details
-  Delete listings that are no longer needed
-  View full details on a separate page
-  Responsive UI built using Bootstrap
-  Form validation implemented using Joi
-  Passport + JWT for authentication
-  Secure authentication using Passport + JWT
-  Protected routes with JWT-based authorization

- Scalable code structure with separate controllers, models, and routes
